在国内如何注册一个免费的ChatGPT账号

一. 准备

ChatGPT现在还没有对国内提供服务，国内的IP访问会被限制，这就需要使用国外的iP进行访问。

在注册ChatGPT账号的过程中，不支持国内手机号注册，需要提供一个国外的手机号，好在目前都有成熟的解决方案，注册的门槛在于解决下面两个问题。

1、代理。代理一定要用美国或欧洲的IP，特别是香港的IP已经不行了。如果你不确定自己的IP是不是美国或欧洲的，http://en.ipip.net/ 查看。

代理我使用的是 沐云 ，用了五六年了非常稳定，可以随意切换iP节点，也可以和家人分享使用，另外新注册用户有3天的免费试用，可以先试用把流程跑通了再决定是否购买。

2、Sms-active接码平台。目前比较好用的就是Sms-active ，使用这个链接注册将会收到第一笔存款中获得10％的现金返还。

二、使用Sms-active接收验证码

1、注册Sms-active账号

打开接码平台 sms-activate.org，注册一个账号

2、然后要充值余额

点击充值，在充值列表中找到支付宝

目前国内用户注册的比较多，Sms-active平台也水涨船高，要求最低支付2美元大概156卢布，能够注册7个账号左右，这个没办法目前这个接码平台最靠谱，学会了可以帮家人朋友注册反正浪费不了。

充值完成后，我们先去注册ChatGPT账号，注册过程中需要手机号验证了，再返回到接码平台获取验证码。

三、注册ChatGPT账号

1、申请一个ChatGPT账号

点击注册账号，如果点击后没有反应，请先判断自己是否能够访问 google ,如果不能访问 google，那你的代理大概率是没有配置好，如果你还没有使用代理，这里推荐使用 沐云 代理，我自己用了五六年了非常稳定，可以随意切换iP节点，也可以和家人共享。

如果你能访问google，但是点击 注册账号 不能正常访问，请把代理的节点选美国切换不同的节点去尝试，使用浏览器的无痕模式进行访问，如果顺利你将会看到下面的界面。

输入自己的邮箱地址，我用的google邮箱成功了，QQ邮箱提示Signup is currently unavailable, please try again later应该就是限制了,其他的没试。

去邮箱里面去查找OPenAI发送的邮箱地址校验，点击Verify email address 按钮会跳转到Tell us about you 的页面，输入First Name 和 Last Name，点击Continue。

这时会让你输入手机号，ChatGPT目前没有对中国开放，不支持中国的手机号，这时要回到第二步的接码平台购买验证码。

2、回到Sms-active购买验证码

  1. 回到接码平台 sms-activate.org ，在左侧找到OpenAI,找到印度尼西亚点击小黄车，目前印度尼西亚的最便宜。

2)在右侧会看到这个页面，把手机号拷贝下来

3)回到ChatGPT的注册流程中，选择左侧的小国旗，选择Indonesia，把手机号拷贝进去

4)点击Send code via SMS，然后去2)页面，去查看获取到的验证码，输入验证码应该就能注册成功了。

3、注册成功，去登录

注册好了ChatGPT 账号，就可以直接登录ChatGPT了，点击ChatGPT网站 跳转登录，如果登录成功大家会看下面这样的界面。

大家看到我的登录界面是ChatGPT Plus，因为我用的比较多对ChatGPT的回答质量也要求比较高，总结ChatGPT Plus比着免费版有3个好处。

1、回答质量更高，免费版有些问题回答是错的，升级Plus后回答的质量更高，GPT4可以更准确地解决难题。

2、对话框能够输入的文字更多，官方说免费版是3000个单词，ChatGPT Plus可以输入25000个单词，能够处理更长的输入内容。

3、ChatGPT Plus不掉线，免费版的时间长不刷新就显示，An error occurred. If this issue persists pleasecontact us through our help center athelp.openai.com。

我个人是ChatGPT的重度用户，GPT4对我来说很重要，它会让我节省很多时间，工作效率提高很多，因为ChatGPT Plus会员不支持国内信用卡支付，想要升级ChatGPT Plus会员，可以看这个教程 ChatGPT Plus开通流程 。

四. 常见问题&解决

1、遇到“Not available, OpenAI‘s service are not available in your country“

解决办法：

把代理的节点切换到美国节点，浏览器使用无痕模式，防止网站获取您的位置信息。

2、遇到”Too Many signups from the same IP”

解决办法：

这个问题一般来说是你的代理服务器商IP库中的IP太少，一个IP太多人注册了，建议换一个好的代理服务器商，如果不想花钱，很多的优秀的代理服务器商是有3天试用的，比如 沐云 。

3、如何切换IP节点？
解决办法：

我使用的 沐云 的代理，它支持ClashX软件切换代理节点，这里是 macOS/苹果电脑 上的使用教程 沐云夏天教程中心 ，Windows/电脑 、Android/安卓、iPhone&iPad、 Linux路由器 上的教程，在沐云夏天教程中心 里切换就可以了。
